# Modern Calculator Web App

A modern, responsive, accessible web-based calculator built with vanilla HTML, CSS, and JavaScript.

## Features

- **Basic Arithmetic**: Addition, subtraction, multiplication, division
- **Advanced Operations**: Percentage, square root, exponentiation, parentheses
- **Calculation History**: Last 50 calculations with localStorage persistence
- **Keyboard Support**: Full numpad and keyboard mapping
- **Responsive Design**: Mobile, tablet, and desktop layouts
- **Dark/Light Theme**: Instant toggle with localStorage persistence
- **Accessibility**: WCAG 2.1 AA compliant, ARIA labels, screen reader support
- **Copy to Clipboard**: Click the display to copy results

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0-9` | Input numbers |
| `+` `-` `*` `/` | Operators |
| `Enter` `=` | Calculate |
| `Escape` | All clear |
| `Backspace` | Delete last digit |
| `.` | Decimal point |
| `(` `)` | Parentheses |
| `%` | Percentage |
| `^` | Exponentiation |

## Tech Stack

- HTML5
- CSS3 (CSS Grid, Flexbox, CSS Custom Properties)
- Vanilla JavaScript (ES6+)
- No external dependencies

## Browser Support
